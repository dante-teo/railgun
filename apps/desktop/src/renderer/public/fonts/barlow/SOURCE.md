# Barlow

- Source: https://github.com/jpt/barlow
- Upstream font version: 1.208
- Source file: `fonts/gx/BarlowGX.ttf` from the upstream `main` archive
- Bundled file: `barlow-upright-variable.woff2`
- License: SIL Open Font License 1.1; see `OFL.txt`

The bundled webfont is the upright, normal-width instance of the upstream
variable source. Its legacy weight coordinates were normalized to the CSS
weight scale, with named instances spanning Thin 100 through Black 900, and
the result was encoded as WOFF2 with FontTools 4.59.1.
